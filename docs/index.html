如何使用

  1. 1. 下载插件，在 VSCode 扩展面板中选择 “从 VSIX 安装”

  2. 2. 菜单 查看-命令面板 或快捷键 Ctrl(Command) + Shift + P，输入 Enable Rainbow Fart

  3. 3. 点击弹出消息的 Open 按钮

  4. 4. 点击授权按钮，并保持网页打开

  5. 5. 回到 VSCode 享受编程吧，尝试输入 function 试一试
